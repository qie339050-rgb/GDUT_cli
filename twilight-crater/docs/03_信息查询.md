# 03 — 信息查询接口

> 所有接口基础 URL：`https://jxfw.gdut.edu.cn`
> 所有接口需要携带有效的 `JSESSIONID` Cookie

---

## 1. 课表查询

### 个人课表主页

```http
GET /xsgrkbcx!xsgrkbMain.action
```

返回 HTML 页面，包含学期选择框和周次选择框。

### 获取全学期课表

```http
GET /xsgrkbcx!xsAllKbList.action?xnxqdm={xnxqdm}
```

| 参数 | 说明 | 示例 |
|------|------|------|
| `xnxqdm` | 学年学期代码 | `202502` |

**返回**：HTML `<table>` 表格
- 列：周一 ~ 周日（有时到周六）
- 行：第1节 ~ 第12节（每2节合并为一大节）
- 单元格内容：`课程名(学分)★周次\n教室`

### 获取指定周次课表

```http
GET /xsgrkbcx!xskbList.action?xnxqdm={xnxqdm}&zc={zc}
```

| 参数 | 说明 | 示例 |
|------|------|------|
| `xnxqdm` | 学年学期代码 | `202502` |
| `zc` | 周次 | `1` ~ `20` |

**返回**：同上，但只显示该周有课的课程。

### 班级课表

```http
GET /xsbjkbcx!xsAllKbList.action?xnxqdm={xnxqdm}&bjdm={bjdm}
```

| 参数 | 说明 | 示例 |
|------|------|------|
| `xnxqdm` | 学年学期代码 | `202502` |
| `bjdm` | 班级代码 | 从页面下拉框获取 |

**返回**：HTML 表格，格式同个人课表。

## 2. 学习计划 / 教学计划

### 计划列表

```http
GET /xsjxjhxx!xsjxjhList.action?lx=01
```

**返回**：HTML 页面，包含教学计划列表表格。

### 查看具体计划

```http
GET /xsjxjhxx!viewJxjh.action?jxjhdm={jxjhdm}
```

| 参数 | 说明 | 示例 |
|------|------|------|
| `jxjhdm` | 教学计划代码 | `1012901` |

### 获取计划课程数据 ⭐

```http
POST /xsjxjhxx!getKcDataList.action?jxjhdm={jxjhdm}
Content-Type: application/x-www-form-urlencoded

(无需额外 body 参数，jxjhdm 在 URL 中)
```

**返回**：JSON 数组，包含计划中的所有课程信息。

## 3. 课程成绩

### 成绩页面

```http
GET /xskccjxx!xskccjList.action?firstquery=1
```

**返回**：HTML 页面（使用 iframe 结构）。

### 获取成绩数据 ⭐

```http
POST /xskccjxx!getDataList.action
Content-Type: application/x-www-form-urlencoded

xnxqdm={xnxqdm}&page=1&rows=100&sort=xnxqdm&order=asc
```

| 参数 | 说明 | 示例 |
|------|------|------|
| `xnxqdm` | 学年学期代码（可选，空=全部） | `202502` |
| `page` | 页码 | `1` |
| `rows` | 每页条数 | `100` |
| `sort` | 排序字段 | `xnxqdm` |
| `order` | 排序方式 | `asc` |

**返回**：JSON

```json
{
  "total": 10,
  "rows": [
    {
      "kcmc": "高等数学",
      "xf": "4.0",
      "zcj": "85",
      "xnxqmc": "2025秋季",
      ...
    }
  ]
}
```

## 通用响应格式

大部分 POST 数据接口返回的 JSON 结构：

```json
{
  "total": 10,
  "rows": [
    { "字段1": "值1", "字段2": "值2", ... },
    ...
  ]
}
```

## 通用请求参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `page` | 页码 | `1` |
| `rows` | 每页条数 | `100`（建议用大值一次取完） |
| `sort` | 排序字段 | 因接口而异 |
| `order` | 排序方式 | `asc` |
| `xnxqdm` | 学年学期代码 | 当前学期 |
