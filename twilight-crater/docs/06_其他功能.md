# 06 — 其他功能接口

---

## 1. 消息通知

### 获取通知列表

```http
POST /notice!getNotice.action
```

**返回**：JSON 通知列表

### 查看公告详情

```http
GET /desktop!ggtzView.action?pk={公告ID}
```

---

## 2. 学习情况概览

通过右侧浮动按钮触发，弹窗展示当前学期的学业概览数据。

```http
GET /xsxxqk!xxqkView.action
```

**返回**：HTML 弹窗内容，包含已修学分、GPA 等汇总信息。

---

## 3. 文档下载

```http
GET /wdxz!wdxzList.action
```

**返回**：HTML 页面，包含可下载文件列表（教务相关表格、申请表等）。

---

## 4. 在线人数

```http
GET /login!getOnlineCount.action
```

**返回**：当前在线人数（JSON 或纯文本）。

---

## 5. 修改密码

```http
GET /xgmm!xgmmView.action
```

修改教务系统本地密码的页面。

---

## 6. 设置

```http
GET /xssz!xsszView.action
```

个人设置页面。
