# 04 — 选课报名接口

> 以下接口基于顶部「选课报名」菜单探索获得。
> 选课是敏感操作，CLI 工具应以 **查询** 为主，操作需谨慎。

---

## 1. 个人选课

### 选课列表页

```http
GET /xsxk!xsxkList.action
```

**返回**：HTML 页面，包含可选课程列表。

### 相关辅助接口

选课模块通常包含以下数据查询接口（遵循系统命名规律）：

```http
POST /xsxk!getDataList.action
Content-Type: application/x-www-form-urlencoded

xnxqdm={xnxqdm}&page=1&rows=100&sort=&order=asc
```

> ⚠️ **注意**：选课和报名类接口涉及数据修改操作（POST 提交选课/退课），使用时需确保参数正确。建议 CLI 工具仅实现查询功能，不实现选课提交。
