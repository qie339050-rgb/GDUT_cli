## 安全机制

### 反爬检测 (reptile.js)

系统加载了 `/static/js/reptile.js`，检测以下特征：

```javascript
// 检测项目
navigator.webdriver           // WebDriver 标志
__driver_evaluate             // Selenium 内部函数
HeadlessChrome               // 无头浏览器 UA
Splash / PhantomJS            // 其他自动化工具
```

**触发后果**：页面重定向，显示 "请不要使用自动化工具访问"

**绕过方式**：
- ✅ `requests.Session`（不执行 JS，完全安全）
- ⚠️ Playwright/Puppeteer 需要 stealth 插件
- ❌ 裸 Selenium 会被直接拦截

### 会话超时

- 登录后 Session 有效期约 **30 分钟**（无操作自动过期）
- 过期后任何接口会返回 302 重定向到登录页
- 检测方式：检查响应 URL 是否包含 `login`
