# 05 — 业务办理接口

> 以下接口基于顶部「业务办理」菜单探索获得。
> 这些接口涉及申请提交，属于写操作，CLI 工具应以查询状态为主。

---

## 1. 缓考申请

```http
GET /xshksq!xshksqList.action
```

查询已提交的缓考申请列表。

---

## 2. 免修申请

GET /xsmxsq!xsmxsqList.action
```

---

## 3. 免听申请

GET /xsmtsq!xsmtsqList.action
```

## 4. 休退学申请

```

## 5. 选订教材

## 6. 成绩复核申请

```

---

## 7. 学籍变动

```http
GET /xsxjbd!xsxjbdList.action
```

查看学籍变动记录。

---

## 8. 创新项目

```http
GET /xscxxm!xscxxmList.action
```

查看创新创业项目。

---

> ⚠️ **通用规律**：每个业务办理模块的数据查询接口通常为：
> ```
> POST /{模块名}!getDataList.action
> Body: page=1&rows=100&sort=&order=asc
> ```
> 返回 JSON 格式。
