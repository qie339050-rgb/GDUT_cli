# 广东工业大学教务系统 (jxfw.gdut.edu.cn) — 接口文档

本文档通过浏览器实际登录并逐页抓包，整理广东工业大学教务管理系统的全部可用接口。

## 文档结构

| 文件 | 内容 |
|------|------|
| [01_系统概览.md](01_系统概览.md) | 系统架构、技术栈、安全机制 |
| [02_登录认证.md](02_登录认证.md) | CAS 登录流程、AES 加密算法、Session 管理 |
| [03_信息查询.md](03_信息查询.md) | 课表、成绩、学习计划、学籍等查询接口 |
| [04_选课报名.md](04_选课报名.md) | 选课、报名相关接口 |
| [05_业务办理.md](05_业务办理.md) | 缓考、免修、免听等申请接口 |
| [06_其他功能.md](06_其他功能.md) | 消息通知、学习情况、文档下载等 |
| [07_参数字典.md](07_参数字典.md) | 学期代码、考试类型等通用参数对照表 |

## 基础信息

- **域名**：`https://jxfw.gdut.edu.cn`
- **认证域名**：`https://authserver.gdut.edu.cn`
- **技术框架**：EasyUI (前端)、Struts2 (.action 路由)
- **会话机制**：JSESSIONID Cookie
- **数据格式**：大部分列表数据通过 POST 返回 JSON，部分页面（如课表）返回 HTML 表格
