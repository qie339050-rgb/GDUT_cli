# 07 — 参数字典

## 学年学期代码 (xnxqdm)

格式：`{年份}{学期号}`

| 代码 | 含义 | 时间范围 |
|------|------|---------|
| `202301` | 2023秋季 | 2023年9月 ~ 2024年1月 |
| `202302` | 2024春季 | 2024年2月 ~ 2024年6月 |
| `202401` | 2024秋季 | 2024年9月 ~ 2025年1月 |
| `202402` | 2025春季 | 2025年2月 ~ 2025年6月 |
| `202501` | 2025秋季 | 2025年9月 ~ 2026年1月 |
| `202502` | 2026春季（当前） | 2026年2月 ~ 2026年6月 |

**规律**：
- 年份后缀 `01` = 秋季（第一学期）
- 年份后缀 `02` = 春季（第二学期）
- 如 `202502` = 2025学年第2学期 = 2026年春季

## 分页参数

所有支持分页的接口通用：

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| `page` | 当前页码 | `1` |
| `rows` | 每页行数 | `100`（一次取完避免分页） |
| `sort` | 排序字段 | 因接口而异 |
| `order` | 排序方式 | `asc` 或 `desc` |

---

## 接口命名规律

系统使用 Struts2 框架，接口遵循 `{模块名}!{方法名}.action` 格式：

| 后缀模式 | 含义 | 示例 |
|----------|------|------|
| `!xxxList.action` | 列表页面（返回 HTML） | `xskccjxx!xskccjList.action` |
| `!getDataList.action` | 数据查询（返回 JSON） | `xskccjxx!getDataList.action` |
| `!xxxView.action` | 详情查看（返回 HTML） | `xsjxjhxx!viewJxjh.action` |
| `!xxxMain.action` | 主页框架 | `xsgrkbcx!xsgrkbMain.action` |

## HTTP 请求通用头

```http
User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36
Accept: text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8
Accept-Language: zh-CN,zh;q=0.9,en;q=0.8
Content-Type: application/x-www-form-urlencoded
Cookie: JSESSIONID={session_id}
```

建议在 `requests.Session` 中设置以上默认请求头，模拟正常浏览器访问。
